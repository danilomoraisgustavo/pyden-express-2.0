Passo a Passo de Configuração J16
1. Inserir o SIM Card

Abra o dispositivo, insira corretamente o SIM Card M2M (com dados habilitados) e certifique-se de que o chip está ativo.

2. Alimentação e Instalação

Conecte o dispositivo à bateria ou parte elétrica do veículo de acordo com o manual. O fio Vermelho (positivo), Preto (negativo), etc.

3. Configurar APN e Servidor

Use os comandos SMS numerados a seguir, enviando cada um deles para o número do SIM Card inserido no dispositivo:

  1. SERVER: SERVER,0,semedcanaadoscarajas.pydenexpress.com,8741,0#
  2. APN: APN,sua.apn,aqui,senha#
  3. TIMER: TIMER,60,3600#

Se preferir usar o IP diretamente, substitua o comando SERVER pelo seguinte:

  1. SERVER: SERVER,0,54.233.187.107,8741,0#
  2. APN: APN,sua.apn,aqui,senha#
  3. TIMER: TIMER,60,3600#
4. Testar Conexão

Após configurar, verifique no sistema se o dispositivo está reportando corretamente a posição (latitude/longitude). O servidor HTTP está na porta 3000, enquanto a porta 8741 é utilizada para os dados do rastreador.

5. Fixação no Veículo

Instale o dispositivo em um local oculto, mas que não bloqueie o sinal GPS.

Observações

Para mais detalhes, consulte o manual do J16 e o protocolo GT06. Em caso de dúvidas, entre em contato com o suporte.
