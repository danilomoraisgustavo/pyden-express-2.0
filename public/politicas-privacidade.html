Políticas de Privacidade

1. Introdução

A PyDen Express está comprometida em proteger a privacidade e os dados pessoais de seus usuários. Esta Política de Privacidade descreve como coletamos, utilizamos, armazenamos e compartilhamos informações dos diferentes perfis de usuários de nosso sistema (incluindo motoristas, fornecedores, fiscais e administradores), em conformidade com a Lei Geral de Proteção de Dados Pessoais (LGPD – Lei nº 13.709/2018) e demais legislações aplicáveis. Ao utilizar os serviços do PyDen Express – seja no painel web, aplicativo móvel ou via API – você concorda com os termos desta Política de Privacidade.

2. Escopo e Aplicabilidade

Esta Política de Privacidade se aplica a todos os usuários e participantes do sistema PyDen Express, abrangendo: (i) motoristas cadastrados na plataforma, (ii) fiscais (inspetores) que realizam atividades de auditoria ou verificação, (iii) fornecedores/parceiros que utilizam o sistema para ofertar serviços ou produtos, e (iv) usuários administradores do sistema. Ela cobre a coleta e o uso de dados pessoais em todos os meios de interação com o PyDen Express, incluindo o portal web administrativo, o aplicativo móvel e quaisquer integrações via serviços de API. Nosso objetivo é garantir que, independentemente do meio de acesso, seus dados pessoais sejam tratados com segurança, respeito e em conformidade com esta política.

3. Dados Coletados e Tratados

Coletamos e tratamos diversos tipos de dados pessoais para viabilizar as funcionalidades do PyDen Express. Esses dados podem variar de acordo com o perfil de usuário, mas de forma geral incluem:

  • Dados de Identificação Pessoal: nome completo, CPF/CNPJ, RG, data de nascimento, endereço, telefone, e-mail, e outras informações de contato. Esses dados são coletados no cadastro de motoristas, fornecedores, fiscais e administradores para identificar o usuário de forma única e permitir comunicações.
  • Documentos e Dados Legais: cópias ou números de documentos oficiais (por exemplo: CNH – Carteira Nacional de Habilitação para motoristas, documentos de identidade, comprovante de endereço, contratos sociais no caso de fornecedores empresariais) e certidões ou antecedentes necessários. Também podemos armazenar termos de consentimento assinados, contratos e registros legais pertinentes à relação do usuário com a plataforma.
  • Dados Veiculares: informações sobre veículos de motoristas (como placa, modelo, ano, cor, número do RENAVAM e documentos do veículo) necessários para cadastro e operação dos motoristas na plataforma. Esses dados garantem a regularidade e identificação dos veículos utilizados nos serviços.
  • Dados de Localização: dados de geolocalização coletados durante o uso do aplicativo pelos motoristas (e possivelmente fiscais), como coordenadas GPS em tempo real, rotas percorridas, horários e locais de partidas e chegadas. A localização é utilizada para monitoramento de rotas, logística das entregas/serviços e segurança das operações em tempo real.
  • Logs e Registros de Atividade: registramos automaticamente eventos e atividades no sistema, tais como data e hora de acessos, endereços IP de conexão, ações realizadas no painel ou aplicativo (por exemplo, criação ou edição de cadastros, envio de formulários, geração de relatórios, login e logout). Esses logs são associados aos perfis (motorista, fornecedor, fiscal, administrador) e servem para fins de auditoria, monitoramento de segurança, solução de problemas e melhoria da plataforma.
  • Dados Financeiros: informações financeiras necessárias para as transações dentro da plataforma. Por exemplo, dados bancários de motoristas e fornecedores (para processamento de pagamentos, repasses ou cobrança de comissões), histórico de pagamentos e recebimentos, valores de corridas/entregas ou serviços realizados, notas fiscais emitidas, etc. Se o sistema realizar pagamentos ou cobranças, podemos armazenar dados como agência e conta bancária, PIX, CNPJ/CPF para faturamento e quaisquer outras informações financeiras estritamente necessárias.
  • Credenciais de Acesso: dados de autenticação como nome de usuário, IDs de usuário e senhas (armazenadas de forma criptografada). Também mantemos tokens de sessão ou identificadores de sessão ativa para gerenciar o login no sistema. Essas credenciais garantem que somente usuários autorizados acessem áreas restritas conforme seu perfil (motorista, fiscal, fornecedor ou administrador).
  • Outros Dados Necessários: quaisquer outros dados pessoais fornecidos voluntariamente pelo usuário ou necessários para funcionalidades específicas do PyDen Express. Por exemplo, preferências do usuário, fotos de perfil, feedbacks e comunicações feitas através do aplicativo. Sempre que novos dados forem coletados, informaremos o usuário sobre a necessidade e a base legal, garantindo transparência no tratamento.

Todos os dados são coletados de forma informada e consentida pelo usuário quando do cadastro ou uso contínuo da plataforma. Apenas solicitamos e armazenamos informações relevantes e necessárias para os fins abaixo especificados.
